Partial function
join' :: [a] -> [a] -> [a] //ทั้งหมดนั้นคือ type ของ function join' โดยมีความหมายว่า [a]/input -> ([a] -> [a])/output โดยมี return type คือ [a] -> [a] 
join' [] ys = ys
join' (x:xs) ys = x : join' xs ys // xs และ ys ต่อคิวเข้า join'
